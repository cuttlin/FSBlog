### AcWing 789.数的范围
#### 题目描述
给定一个按照升序排列的长度为n的整数数组，以及 q 个查询。

对于每个查询，返回一个元素k的起始位置和终止位置（位置从0开始计数）。

如果数组中不存在该元素，则返回“-1 -1”。
#### 输入格式
第一行包含整数n和q，表示数组长度和询问个数。

第二行包含n个整数（均在1~10000范围内），表示完整数组。

接下来q行，每行包含一个整数k，表示一个询问元素。
#### 输出格式
共q行，每行包含两个整数，表示所求元素的起始位置和终止位置。

如果数组中不存在该元素，则返回“-1 -1”。
#### 数据范围
1≤n≤100000
1≤q≤10000
1≤k≤10000
#### 输入样例
~~~~
6 3
1 2 2 3 3 4
3
4
5
~~~~
#### 输出样例
~~~~
3 4
5 5
-1 -1
~~~~
#### 题目思路
采用二分查找算法
```C++
#include<iostream>
using namespace std;
const int N = 1e5+10;
int q[N],n,m;


int main()
{
    scanf("%d%d",&n,&m);
    for(int i=0;i<n;i++)scanf("%d",&q[i]);
    int x;
    while(m--)
    {
        scanf("%d",&x);
        int l=0,r=n-1;
        while(l<r)
        {
            int mid=l+r>>1;
            if(q[mid]>=x)r = mid;    //大于等于缩小右边界，所以此循环求得左边界
            else l = mid+1;
        }
        if(q[l]!=x)
        {
            printf("-1 -1\n");
            continue;
        }
        printf("%d ",l);
        l = 0,r = n - 1;
        while(l<r)
        {
            int mid = l+r+1>>1;
            if(q[mid]<=x)l = mid;    ///小于等于缩小左边界，所以此循环求得右边界
            else r = mid - 1;
        }
        printf("%d\n",l);
    }
    return 0;
}
```
